### 🏭 SupplyMind
**Agentic AI OS for Supply Chain Intelligence**

An LLM-orchestrated multi-agent platform that autonomously manages global supply chains — demand forecasting, supplier risk scoring, inventory rebalancing, logistics optimization, and sustainability tracking, all coordinated through a hierarchy of 8 specialized AI agents.

**What makes it different:**
- `Orchestrator` dynamically routes natural language queries to the right agents
- Supplier risk scoring runs through a **trained scikit-learn ML pipeline** — fully bypassing the LLM for deterministic, explainable predictions
- **Smart mock-fallback**: all 12 LangChain tools execute on real data even when Ollama is offline
- Real-time agent status polling every 3s; inter-agent message bus with ring-buffer logging

**8 Agents · 12 Tools · 10 Workflows · MongoDB · FastAPI · React 19**

`FastAPI` `LangChain` `LangGraph` `Ollama (llama3.2)` `MongoDB` `React 19` `Tailwind` `scikit-learn`

### ⚖️ GovernAI
**Responsible AI Compliance Platform via RAG**

Upload an organizational policy or document → select regulatory frameworks → receive structured compliance assessments with per-article risk levels, metric scores, and actionable recommendations — grounded in actual regulatory text via RAG.

**What makes it different:**
- **FAISS + LangChain LCEL** retrieves exact regulatory passages per framework (DPDP · GDPR · EU AI Act · NIST AI RMF)
- Analysis runs **asynchronously** in the background; users fill the governance questionnaire in parallel
- Scores computed manually from per-article results — eliminates LLM arithmetic hallucinations
- Full SQLite-backed audit trail with 10+ distinct event types at every pipeline stage

**Dual LLM · FAISS · Async Analysis · Governance Module · Full Audit Trail**

`FastAPI` `LangChain (LCEL)` `FAISS` `Ollama / Gemini 1.5` `React 18` `SQLite` `nomic-embed-text`
